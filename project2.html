Robot Fleet Management System

A scalable and intuitive system to simulate and manage the coordination of a cleaning robot fleet.

Overview

The Robot Fleet Management System is a scalable and intuitive platform designed to simulate and manage the coordination of a cleaning robot fleet. It consists of four key components: a Graphical User Interface (UI) tailored for different user roles (Building Operator, Building Manager, Field Engineer, Senior Manager), a System Manager that synchronizes operations, a Status Database for storing real-time robot and job information, and a Simulation Engine to mimic real-world robot activities.

Key Features

  • Add and manage floors, robots, and tasks with ease
  • Real-time simulation with live updates
  • Seamless switching between user roles
  • Role-specific functionalities tailored to user responsibilities
  • Monitor robot performance and task progress through an intuitive dashboard

Technologies Used

Link to think

A modern and responsive website built with pure HTML/CSS and JavaScript to connect professor and students for research opportunities.

View live website

Overview

The link to think website is designed to connect students with research opportunities and professors with potential student researchers at University of Richmond. Built with HTML, CSS, and JavaScript, the platform facilitates communication and provides an organized interface for displaying research openings/student researchers, submitting and tracking applications and enabling messanging between professor and students.

Key Features

  • Login and sign-up pages
  • Research openings/student researchers pages featuring a search bar, advanced filters, and detailed result listings
  • Application page and a Past Applications page (student users)
  • Messaging system allowing direct communication between professors and students
  • Intuitive navigation system
